About Me

Although I would consider my studies at NSS to be one of the most difficult endeavors I've taken part in, it has also been the most rewarding. Around the Web

VYNL

Using AngularJS, Bootstrap, Firebase & the LastFM API this app allows users to keep track of their current record collection, find nearby record stores and get recommendations for future purhases. The idea for this came out of my tendencies to buy a vinyl record in-store only to later find out that it is already in my collection. The app creates a simple & mobile friendly user experience for use on-the-go!

  • Date: September 2017
  • Main Technologies Used: AngularJS, Bootstrap, Firebase & the LastFM API


Horse Haven

Created during the Code for a Cause hackathon, this app is a solution to the needs of Horse Haven of Tennessee. Myself and 2 others were able to come together over the course of a couple days to build a ready-to-use, internal application to aid in the everyday management and report creation for an extremely busy non-profit.

This was the winning app in the competition and I am extremely proud to be a part of something that is actively making an organization's day-to-day operations more efficient.

  • Client: Horse Haven of Tennessee
  • Date: October 2017
  • Main Technologies Used: AngularJS, Bootstrap, Firebase


Scramble

Scramble is built with a Ruby on Rails API & SQLite3 database to feed an AngularJS frontend and helps golfers connect with area competitions. Users can create their own competitions for others to view. Additionally, using Leaflet JS, users are able to view available competitions on a map & click through for signup.

  • Date: December 2017
  • Main Technologies Used: Ruby, Ruby on Rails, AngularJS, Bootstrap 4, SQLite, LeafletJS
